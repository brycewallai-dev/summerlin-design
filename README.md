## How to continue this with Claude Code
1. Open a terminal, navigate to this folder
2. Run `claude` to start Claude Code in this directory
3. Tell it what you want next, e.g.:
   - "Build residential.html, portfolio.html, about.html, and contact.html using the same design system as commercial.html"
   - "Deploy this to Vercel"
   - "Add these images to the hero section: [drag in image, tell it the filename]"
4. Claude Code will handle the code — you just describe what you want in plain English

## Deploying
Easiest path: ask Claude Code to "deploy this site to Vercel" — it will walk through connecting a GitHub repo and getting a live URL, typically in a few minutes.
